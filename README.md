# 🎯 Quiz Game Application

An interactive multiple-choice quiz game built using **HTML, CSS, and JavaScript**.  
The application displays questions, allows users to select answers, and shows the final score at the end.

---

## 🚀 Features

- Multiple-choice questions
- Interactive UI
- Score calculation
- Real-time answer selection
- Clean and responsive design

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
